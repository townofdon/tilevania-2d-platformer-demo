# Unity WebGL Build - Docker

This docker setup runs a unity webgl build locally.

## Getting started

First, run:

```
docker-compose up
```

Then, open up the following url in your browser:

```
localhost://8080
```

To stop an instance:

```
docker-compose down
```

## Config

The `webgl.conf` file is an nginx config file.
